## Getting Started

First, run the development server:

Deploy Locale : Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
```bash
npm run dev
```

Deploy Github Pages tramite .github/main.yml:
```bash
npm run deploy
```
